
# Workspace Instructions

When the user sends a message starting with a `/` command, read `origin.md` and follow the **Commands** section to route and execute the command. Treat `aios/` as the AIOS artifact root for `plan.md`, `state.md`, `planning/`, `development/`, `enhancements/`, `plans/`, `proposals/`, `reflections/`, and `review/` unless a command explicitly says otherwise.

Current commands:
/birth
/continue
/enhance
/document
/steer
/fix
/question
/reflect
/propose
/sprint
/envision
/implement

To find the list of recognized commands, read the `## Commands` section in `origin.md` — each `### /command` heading is a valid command.
