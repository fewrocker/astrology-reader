
# Workspace Instructions

When the user sends a message starting with a `/` command, read `origin.md` and follow the **Commands** section to route and execute the command. Make sure you actually find the command and the instructions to work with it before starting.

Current commands:
/birth
/continue
/enhance
/document
/steer
/fix
/question
/reflect
/propose
/envision
/implement

To find the list of recognized commands, read the `## Commands` section in `origin.md` — each `### /command` heading is a valid command.
