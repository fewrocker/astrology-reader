Build an astrology website for daily birth chart readings. It has to be very thorough and state of the art, and beat the readers that you have online. Meaning it should ACTUALLY do astrological calculations to get all the insights. The user will answer an interactive form of a few steps to say date of birth, time of birth, city of birth (needs to actually implement a cities api for autocomplete), then a few questions on what the person wants to focus on the reading. after that, do the calculations, fetches on apis, anything you need to return the results with a complete chart drawn with all the planets, aspects, interactions, and then actually give a good interpretation of what it means (maybe prompt gpt if you need or research the best way to find these interpretations). Please make a concise interpretation first, and then jump in the details below. make a nice mystic ui but not too over the top
